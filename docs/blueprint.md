# **App Name**: NEU LabTrack

## Core Features:

- Professor Institutional Login: Secure authentication for professors using Google accounts, restricted to @neu.edu.ph institutional emails, with account blocking capabilities managed by admins.
- Professor Session Tracking: Allows professors to input and initiate lab usage sessions by providing essential details such as room number, college, program, section, and start time, followed by a confirmation message.
- Professor Session Completion: Professors can explicitly end their active sessions, automatically triggering the calculation and logging of the total usage duration.
- Admin Usage Dashboard: Provides administrators with a comprehensive dashboard to view all usage logs, offering search functionality, filtering options (daily, weekly, monthly, custom dates), and key usage statistics (total rooms, hours used, most utilized rooms/professors, active sessions).
- Admin Room Management: Enables administrators to efficiently add, edit, or delete laboratory rooms, ensuring the system accurately reflects the available lab resources.
- Admin Account Management: Allows administrators to manage professor accounts, specifically by blocking or unblocking access, to maintain system security and authorization.
- Admin Report Generation: Facilitates the export of filtered usage data (by professor, room, or date range) into CSV or Excel formats for detailed documentation and external analysis.

## Style Guidelines:

- Light color scheme with a prominent primary blue (Hue: 220, Saturation: 75%, Lightness: 55%) derived from the NEU branding. RGB Hex: #266AFF.
- Background color is a very light, cool grey with a hint of blue, to provide a clean and subtle base. RGB Hex: #F7F8FC.
- Accent colors from the NEU branding: a vibrant yellow (RGB Hex: #FFD43D) and an energetic green (RGB Hex: #4BC64F) for highlights and status indicators.
- The font 'Inter' (sans-serif) will be used for both headlines and body text, providing a modern, neutral, and highly readable appearance suitable for data-heavy content.
- Use clean, modern, outline-style icons with clear visual communication for navigation, actions, and data representation.
- A well-organized dashboard layout featuring a fixed sidebar for primary navigation and a flexible main content area, utilizing cards for statistical overviews and clean data tables for logs and lists.
- Subtle and smooth animations will be applied to UI element state changes and page transitions, ensuring a fluid and pleasant user experience without distractions.
